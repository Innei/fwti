# FWTI — 恋爱废物人格测试

### Fèiwù Type Indicator · v0.4

> 1 道前置题 + 20 道共享主干 + 1 道共享彩蛋 + 四状态专属扩展（7–9 题 / 状态）+ 动态 follow-up 子题，随状态分流；全程 30–39 题。10 个叠加隐藏标签 + 10 种隐藏人格，四大维度交叉分析，精准匹配你在恋爱中的废物类型。
> 测试仅供娱乐，请勿用于相亲、挽回、分手、发律师函或自我攻击。
> 虽然是纯娱乐测试，但四大维度的理论锚都有真·心理学文献出处（见文末「学术依据」），所以你看似被骂，其实是被 Fraley 老师骂。

## 〇、前置题（不计分，仅用于题目措辞路由）

**第 0 题**
你目前的恋爱状态是？

- A. 有对象/正在恋爱
- B. 暧昧中/有心动的 TA
- C. 单身，但心里藏着某个人/念念不忘的历任
- D. 纯单身，古生物级别的自由人

> 系统会根据你的选择，把后续题目里的主语"TA"投射到合适的对象身上（B 选项对应"你暧昧的 TA"，C/D 选项对应"心里/假想中的那位"）。维度计分完全一致，只是措辞不同，单身选手也不会做到一半出戏。

## 一、四大维度

| 维度         | 极性 A                        | 极性 B                         | 理论锚（严肃版）                                                       | 核心问题                   |
| ------------ | ----------------------------- | ------------------------------ | ---------------------------------------------------------------------- | -------------------------- |
| **主动性**   | **G（Go·冲）** — 主动出击型   | **D（Dwell·蹲）** — 原地等死型 | 接近–回避社交动机（Gable, 2006）/ BAS 行为激活系统（Carver & White, 1994） | 你在恋爱里是冲锋还是蹲坑？ |
| **情绪表达** | **Z（Zha·炸）** — 情绪外放型  | **R（Ren·忍）** — 情绪内收型   | 情绪调节策略：表达 vs 抑制（Gross & John, 2003 ERQ）                   | 有事你是炸了还是忍了？     |
| **亲密需求** | **N（Nian·黏）** — 高度黏合型 | **L（Li·离）** — 独立疏离型    | 成人依恋"回避"维度（Brennan, Clark & Shaver, 1998; Fraley et al., 2000） | 你是黏人精还是独处王？     |
| **安全感**   | **Y（Yi·疑）** — 患得患失型   | **F（Fo·佛）** — 佛系信任型    | 成人依恋"焦虑"维度（同上）                                             | 你是疑心病还是佛祖？       |

四个字母组合 → 2⁴ = **16 种恋爱废物人格 + 10 种隐藏人格**（ALL + RAT + PURE + MAD + E-DOG + CHAOS + CPU + BENCH + VOID + LIMBO）

> 如果你在多个维度得分完全平票，你不会被判为"更废的那一端"——会直接归入隐藏人格「**我全都要（ALL）**」。详见第五节计分规则。v0.3 已新增 7 种独立隐藏人格（RAT / PURE / MAD / E-DOG / CHAOS / CPU / BENCH）；v0.4 再增 2 张状态门禁人格：**VOID · 电子断联户**（仅 solo 可解）与 **LIMBO · 意难平学家**（仅 crush 可解）。各有独立触发条件，优先级高于 16 格判定与 ALL。

## 三、隐藏叠加标签（不改变四字母，只在结果页加彩蛋）

在回答主线题的过程中，系统会暗中累积以下标签的触发点。满足条件的玩家会在结果页看到一条小字："恭喜你还额外解锁了 XX 标签"：

v0.4 触发条件均以 **语义锚 + 维度 ratio** 表达，避免与具体题 id 强耦合（题库扩展时不需要同步改 scoring）。语义锚由 `src/logic/semanticIds.ts` 统一维护，表中仅列人话描述，具体题 id 见该文件。

| 标签                 | 触发条件（简述）                                                   |
| -------------------- | ------------------------------------------------------------------ |
| **撤回大师**         | 共享彩蛋 Q31 选 A（纠结值 ≥ 2，`SEMANTIC.REVOKE_RETRY`）           |
| **夜谈冠军**         | 三条"情绪外放 / 脑补安全感"题（受委屈、哭、脑补不爱我）中 ≥ 2 题选极端端 |
| **朋友圈考古学家**   | "翻聊天记录" + "点赞异性"两道 YF 题皆选 A（`JEALOUSY_EX_STALK` + `LIKE_OTHER_GENDER`） |
| **薛定谔的前任**     | `status === 'crush'` + "临别爆发"题 + "先脑补被抛弃"题皆选极端 A（`FAREWELL_BLAME` + `BAD_NEWS_PANIC`，两题均落入 extensions.crush） |
| **电子乙方**         | 约会地点让 TA 定 + 吵架先低头 + 2 小时没回先追一条（`DATE_LOCATION_DEFER` + `COLD_WAR_APOLOGIZE` + `TWO_HOUR_NOREPLY`）|
| **人形 ATM**         | Q33（经济型，extensions.dating）或 Q34（情绪客服型，extensions.crush）任一选 A |
| **空想家**           | `status === 'solo'` + 路径上极端选项占比 ≥ 40%（单身也能单方面废物）|
| **典中典**           | 四维 ratio 绝对值皆 ≥ 0.9（四维同步压顶）                          |
| **普信选手**         | 判定为 GRLF（正常人 SANE）+ 三道 YF 佛系题皆选 C（`CALM_RIVAL` + `BAD_NEWS_PANIC` + `OTHER_PRAISE` 三者 polarity 均为 -1）|
| **退退退**           | 答题过程中改答 ≥ 3 次。前端 quiz 页累计，不入 hash，分享链接观众端恒为 0 |

> 这些标签都是"与"关系的叠加触发，实现很便宜，但结果页的惊喜感和传播点会多不少。

## 四、题库结构（分支化，30–37 题）

> **v0.4 结构**：META 前置题（Q32，独立）→ 共享主干（20 维度题 + 共用彩蛋 Q31）→ 状态扩展段（按 META 分流 dating / ambiguous / crush / solo，各 7–9 题）→ 动态 follow-up 子题（按扩展题选项触发，递归深度硬上限 2）。单路径实际作答约 30–37 题。
>
> **措辞**：下文 `TA` 指"你在意的那个人"——有对象即对象，暧昧中即暧昧对象，单身藏着谁即心里的那位，古生物单身即假想中的 TA / 相亲对象 / 交友软件上你滑右的脸。
>
> **归一化**：程度类题 5 档（±2/±1/0），情境角色化题 3 档（+2/0/-2）。评分按 `option.score / 2` 归一为 [-1, +1] 单题贡献，再按维度"路径题数"取均值 → `ratio`，与题量解耦。
>
> **ID 约定**：题 id 全局唯一、append-only。Q1–Q30 中仅 20 题保留为主干（其余 10 题 v0.4 裁汰）；Q31 为共享彩蛋；Q32 为 META；Q33 / Q34 为原 ATM 彩蛋分别迁入 dating / crush extension，id 保留；Q35 为 v0.4 新增 solo 彩蛋（VOID 证据题）；Q36 起为新增扩展题与 follow-up。

## 五、计分规则（v0.4 · ratio 归一化）

**单题归一**：每题的 `option.score` 先除以 2，得到一个落在 [-1, +1] 的单题贡献（3 档 A=+1 / B=0 / C=-1；5 档 A=+1 / B=+0.5 / C=0 / D=-0.5 / E=-1）。此步骤抹平 3 / 5 档之间的档位差。

**维度比值（ratio）**：对当前路径上属于某维度的所有题（主干 + 该状态 extension + 已触发的 followup），取其归一化贡献的算术平均，得到该维度的 `ratio ∈ [-1, +1]`。正向指极性 A（G / Z / N / Y），负向指极性 B（D / R / L / F）。`ratio === 0` 即两极严格持平。

- **与题量解耦**：主干 + extension + followup 的题数因 status 和 follow-up 触发而不同，均值消除了题量差异对维度判定的污染。
- **最小题数保底**：每维至少有 4 题有效贡献才参与判定（未达标则该维度按 0 处理，交由 ≥2 维平票兜底 → ALL）。主干段每维恰好 5 题共享，全状态天然达标。

**16 格判定**：对四维分别取 `sign(ratio)`：> 0 → 极性 A（G/Z/N/Y），< 0 → 极性 B（D/R/L/F）。

**ratio === 0 的处理（v0.4 修订）**：

- 仅 1 个维度严格 `ratio === 0`：v0.3 起已不再展示星号，直接硬分入该维度的默认"废端"（GD → D，ZR → Z，NL → N，YF → Y）。此规则保持不变。
- ≥ 2 个维度严格 `ratio === 0`：归入隐藏人格「**我全都要（ALL）**」。

> 从依恋研究角度看，"外放"和"内收"、"黏"和"离"都各有各的废法，简单把一端判为更废并不公平；但为了保留测试的娱乐感，又不能让每个平票都变成正常人，所以保留 ALL 作为兜底出口。v0.3 已取消星号展示但保留 ALL，v0.4 沿用。
>
> 默认废端的符号约定有意不对称：**GD 默认 D（蹲）**，其他三维默认 A（Z、N、Y）。因为"蹲"是 GD 维的废端，而 Z / N / Y 才分别是 ZR / NL / YF 的废端——四个维度都偏向"更戏剧化的一侧"。此约定与 v0.3 相同，scoring.ts 中有注释块标记。

### 五·附 · 隐藏人格判定顺序（v0.4 更新）

在四字母归格之前，系统先按下列顺序检查特殊隐藏条件。命中任一条即判为该隐藏人格，**不再回落至 16 格**。v0.4 触发条件全部改用"ratio"（每维已归一化比值，`-1 ≤ ratio ≤ +1`，正向指极性 A、负向指极性 B），不再依赖绝对分或具体题 id，以此与题库任意扩展解耦：

1. **MAD · 发疯文学家** — `ratio.GD ≥ 0.9 && ratio.ZR ≥ 0.9 && ratio.NL ≥ 0.9 && ratio.YF ≥ 0.9`（四维几近压顶 GZNY）
2. **RAT · 鼠鼠恋人** — `status === 'solo' && ratio.GD ≤ -0.6 && ratio.ZR ≤ -0.4 && ratio.NL ≤ -0.4 && ratio.YF ≥ 0.8`
3. **PURE · 纯爱战士** — `(status === 'solo' || status === 'crush') && ratio.GD ≥ 0.8 && ratio.NL ≥ 0.6 && ratio.YF ≤ -0.6`
4. **CPU · CPU 恋人** — `ratio.YF ≥ 0.8 && polarityOf(SEMANTIC.IMAGINE_NOT_LOVED) > 0 && polarityOf(SEMANTIC.BAD_NEWS_PANIC) > 0`
5. **CHAOS · 已读乱回** — `(ratio.YF ≤ -0.4 || ratio.NL ≤ -0.4) && lens.polarityVariance('ZR') ≥ 2`（一维内自相攻伐）
6. **E-DOG · 赛博舔狗** — `Math.abs(ratio.NL) ≤ 0.2 && polarityOf(SEMANTIC.TWO_HOUR_NOREPLY) > 0 && polarityOf(SEMANTIC.POST_DATE_TEXT) > 0`
7. **BENCH · 备胎之王** — `status === 'ambiguous' && ratio.GD ≥ 0.2 && ratio.ZR ≤ -0.4 && ratio.NL ≤ -0.2 && ratio.YF ≥ 0.4 && polarityOf(SEMANTIC.LIKE_OTHER_GENDER) > 0`
8. **VOID · 电子断联户** — `status === 'solo' && ratio.GD ≤ -0.8 && ratio.NL ≤ -0.6 && polarityOf(SEMANTIC.OFFLINE_RELIEF) > 0`（v0.4 新增；见 Q35）
9. **LIMBO · 意难平学家** — `status === 'crush' && ratio.YF ≥ 0.8 && ratio.NL ≥ 0.3 && polarityOf(SEMANTIC.JEALOUSY_EX_STALK) > 0`（v0.4 新增；查岗博士后）
10. **ALL · 我全都要** — ≥ 2 个维度的 `ratio` 严格等于 0（原 LIMBO 规则，置最末作为兜底）

若无任一命中，进入正常 16 型判定流程（§五平票规则）。

> 设计意图：v0.3 之前隐藏人格仅 LIMBO 一张，承载"两端打平"这一种处境；v0.3 把"废得极致"、"废得偏狭"、"废得分裂"等多种处境也分门别类地接住。v0.4 再追加两张 **状态门禁** 人格：VOID 专接"单身且主动彻底熄火、线上失踪"的玩家，LIMBO 专接"暗恋但查岗成瘾、把 TA 的朋友圈考古成论文"的玩家。状态门禁避免与老玩家现存 v0.3 结果冲突。优先级顺序按"越具体越靠前"：MAD 的触发条件最窄，ALL 的最宽（兜底）。

## 八、学术依据（虽然是娱乐测试，但维度有真出处）

FWTI 的四个维度并非完全瞎编，它们分别对应心理学里研究浪漫关系最常用的几个量表。下列文献既是维度设计的锚点，也是未来"认真版解读"可以引用的来源：

**成人依恋理论（Adult Attachment Theory）—— 对应 Y/F 与 N/L**

- Hazan, C., & Shaver, P. (1987). Romantic love conceptualized as an attachment process. *Journal of Personality and Social Psychology, 52*(3), 511–524.
  → 首次把 Bowlby 的依恋理论应用到成人浪漫关系，提出"焦虑 / 回避"两个核心维度。
- Bartholomew, K., & Horowitz, L. M. (1991). Attachment styles among young adults: A test of a four-category model. *Journal of Personality and Social Psychology, 61*(2), 226–244.
  → 四类依恋模型（安全 / 专注 / 回避–拒绝 / 恐惧–回避），与 FWTI 四字母空间存在部分同构关系。
- Brennan, K. A., Clark, C. L., & Shaver, P. R. (1998). Self-report measurement of adult romantic attachment: An integrative overview. In J. A. Simpson & W. S. Rholes (Eds.), *Attachment theory and close relationships* (pp. 46–76). Guilford Press.
  → 提出 ECR 量表，把依恋简化为"焦虑"和"回避"两个连续维度——本测试 N/L 和 Y/F 的原型。
- Fraley, R. C., Waller, N. G., & Brennan, K. A. (2000). An item response theory analysis of self-report measures of adult attachment. *Journal of Personality and Social Psychology, 78*(2), 350–365.
  → ECR-R 修订版，本测试 N/L 和 Y/F 维度的题目灵感大量参考此量表的条目。

**情绪调节（Emotion Regulation）—— 对应 Z/R**

- Gross, J. J., & John, O. P. (2003). Individual differences in two emotion regulation processes: Implications for affect, relationships, and well-being. *Journal of Personality and Social Psychology, 85*(2), 348–362.
  → ERQ 量表，区分"认知重评"与"表达抑制"两种情绪调节策略，对应本测试的 Z/R 维度。

**接近–回避动机（Approach–Avoidance Motivation）—— 对应 G/D**

- Carver, C. S., & White, T. L. (1994). Behavioral inhibition, behavioral activation, and affective responses to impending reward and punishment: The BIS/BAS Scales. *Journal of Personality and Social Psychology, 67*(2), 319–333.
  → BIS/BAS 量表，提供 G/D 维度的生物动机基础。
- Gable, S. L. (2006). Approach and avoidance social motives and goals. *Journal of Personality, 74*(1), 175–222.
  → 亲密关系情境下的接近 / 回避目标，对应 G/D 在恋爱里的具体表现。

**中文版量表（如果你真的对自己好奇）**

- 李同归, 加藤和生. (2006). 成人依恋的测量：亲密关系经历量表（ECR）中文版. *心理学报, 38*(3), 399–406.

> **郑重声明**：FWTI 借用了这些理论的"维度结构"与部分题目灵感，但**并未经过心理测量学的信效度验证**。它不是诊断工具、不能替代 ECR-R，也不能用来给自己或别人贴标签。如果看完结果你真的对自己的依恋模式感到好奇，建议去做一份正规的 ECR-R 中文版，或找靠谱的心理咨询师聊聊。被一个网络测试逗笑是好事，被它说服就不是了。
