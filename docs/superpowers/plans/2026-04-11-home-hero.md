### Task 2: `.home-subtitle` 与响应式字号

**Files:**

- Modify: `src/global.css` — 在 `.home-title` 与 `.home-lede` 之间（约 441 行后）插入 `.home-subtitle`；在 `@media (max-width: 900px)`、`720px`、`460px` 中为 `.home-subtitle` 设置递减 `font-size`，避免窄屏副标题喧宾夺主。

- [ ] **Step 1: 桌面基础样式**

在 `src/global.css` 中 `.home-title { ... }` 块之后、` .home-lede` 之前追加：

```css
.home-subtitle {
  font-family: var(--fwti-font-title);
  font-weight: 600;
  font-size: 22px;
  line-height: 1.35;
  color: rgba(255, 255, 255, 0.92);
  margin: 0 auto 20px;
  max-width: 640px;
  letter-spacing: -0.01em;
}
```

将 `.home-title` 的 `margin-bottom` 从 `22px` 调整为约 `12px`–`16px`，使两行主标题与副标题间距紧凑但仍可分層（具体像素以实现时视觉为准，以「副标题仍明显小于 h1」为判据）。

- [ ] **Step 2: 响应式**

在现有 `@media (max-width: 900px)` 内 `.home-title { font-size: 44px; }` 之后追加：

```css
  .home-subtitle {
    font-size: 19px;
  }
```

在 `@media (max-width: 720px)` 内 `.home-title { font-size: 36px; }` 之后追加：

```css
  .home-subtitle {
    font-size: 17px;
  }
```

在 `@media (max-width: 460px)` 内 `.home-title { font-size: 30px; }` 之后追加：

```css
  .home-subtitle {
    font-size: 15px;
  }
```

- [ ] **Step 3: Commit**

```bash
git add src/global.css
git commit -m "style(home): subtitle typography for new hero h1"
```
